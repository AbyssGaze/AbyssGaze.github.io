环境搭建

搭建好[Hexo](https://hexo.io/docs/)的博客，需要依赖于Node.js和Git，所以在安装hexo前需要先安装Node.js和Git，

1.安装 Node.js

[Node.js]，(https://nodejs.org/en/)下载的是v6.9.4 LTS版本，一路点下来就完美地完成安装。

2.安装 Git

默认看到此篇博客的已经在电脑上安装了git，当然，没有安装的童鞋可以安装homebrew然后安装git，或者直接安装Xcode，在此介绍homebrew下安装git的方法。
